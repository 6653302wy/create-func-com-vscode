# create-func-com （ts 语言）

1. 一键生成 react 函数式组件模板代码

在 vscode 资源管理器中，选中需要创建组件的根文件夹，右键菜单选择“创建函数式组件”选项

生成的代码结构：

```
ComponentName
└───assets
│     index.tsx
│     styles.css
```

2. 一键生成 react 页面模板代码

创建方式同上，右键菜单选择“创建页面”

生成的代码结构：

```
page-name
└───page
|     └─index
|         └─components
│             └─App
│               └─assets
│               | index.tsx
│               | styles.css
│
│   index.html
│   index.tsx
```
